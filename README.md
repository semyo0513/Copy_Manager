# 🚀 모두의 복붙 (Copy Manager)
> **전국 초·중·고등학교 선생님들을 위한 나이스(NEIS) 생기부 세특 초고속 자동 입력 프로그램**

본 프로그램은 매 학기 말 생활기록부(생기부) 및 과목별 세부능력 및 특기사항(세특)을 입력하느라 겪으시는 전국 선생님들의 업무 피로도를 제로로 만들기 위해 기획 및 제작된 데스크톱 업무 자동화 매크로 프로그램입니다.

엑셀에서 수많은 학생의 세특 내용을 복사한 뒤, 클릭 한 번으로 나이스 웹 화면의 학생별 입력창에 순차적으로 붙여넣고 자동으로 탭(Tab) 이동을 수행하여 단순 반복 작업을 완전히 자동화합니다.

---

## ✨ 핵심 특장점 및 기능

### 1. 엑셀 다중 복사 자동 인식 & 파싱
* 엑셀에서 입력 대상 학생들의 세특 영역(여러 행과 열)을 통째로 드래그하여 복사(`Ctrl + C`)하기만 하면, 프로그램이 실시간으로 클립보드를 감지하여 각 셀 단위 데이터를 완벽히 분리 및 대기 상태로 로드합니다.

### 2. 한글 깨짐 및 오타 걱정 없는 복사-붙여넣기 방식
* 텍스트를 한 글자씩 타이핑하는 타자 매크로 형식이 아닙니다. 클립보드 버퍼에 데이터를 한 글자씩 매핑하여 시스템 단축키(`Ctrl + V`)로 고속 입력하므로, **영타 변환 오류, 한글 초성/종성 분리 및 글자 밀림 오타가 100% 발생하지 않습니다.**

### 3. 초정밀 전역 긴급 정지 (Kill Switch)
* 나이스(NEIS) 사이트의 반응 속도 지연 등으로 입력창 포커스가 엉뚱한 곳으로 튀었을 경우, 프로그램 창을 누르지 않아도 키보드의 **`ESC`** 또는 **`F12`** 키를 누르면 **0.05초 내에 동작이 즉시 강제 중단**되어 매우 안전합니다.

### 4. 나이스 맞춤형 지연 속도(Delay) & 탭 이동 제어
* 학교 컴퓨터 사양 및 교육청 서버 반응 지연에 최적화할 수 있도록 각 셀 입력 후 다음 칸으로 넘어가기 전의 **대기 속도(0.1초 ~ 2.0초)**를 슬라이더로 쉽게 미세 조정할 수 있습니다.
* 한 학생의 입력 창에서 다음 학생의 입력 창으로 가기 위한 **탭(Tab) 키 이동 횟수**를 터치 스피너를 이용해 손쉽게 세팅할 수 있습니다.

### 5. 트렌디한 모던 UI/UX
* 웹 프론트엔드 기술(Eel + HTML5/CSS3/JS)을 결합하여 기존 칙칙한 매크로 프로그램과 차별화된 **세련된 다크 테마**와 **부드러운 네온 글로우 펄스 애니메이션**, **바운스 클릭 피드백**, **실시간 진행률 프로그레스 바** 및 **Toast 알림**을 제공합니다.

## 💻 사용 방법

1. **데이터 복사**: 엑셀(Excel)에서 입력할 학생들의 세특 데이터 범위를 드래그하여 복사(`Ctrl + C`)합니다.
2. **프로그램 실행**: `모두의_복붙.exe` 파일을 실행합니다.
   * *※ 전역 핫키(ESC) 인식 기능 활성화를 위해 **[마우스 우클릭] ➔ [관리자 권한으로 실행]**을 권장합니다.*
3. **설정값 지정**: 
   * 입력된 값 확인 후 다음 칸으로 넘어갈 **탭(Tab) 이동 횟수**를 조절합니다. (일반적으로 나이스 세특 창은 1회 또는 2회)
   * 나이스 서버 반응 속도를 고려하여 **입력 지연 속도(Delay)**를 세팅합니다. (기본 0.2초 권장)
4. **자동 입력 작동**:
   * **[자동 입력 시작]** 버튼을 누릅니다.
   * **3초의 카운트다운 시간** 동안 나이스 웹 화면의 **첫 번째 학생의 입력 칸(텍스트 에어리어)**을 마우스로 신속히 클릭하여 커서(포커스)를 위치시켜 둡니다.
5. **완료 및 정지**: 카운트다운이 완료되면 자동으로 고속 입력 및 탭 이동 루프가 작동하며, 중지하고 싶을 때는 즉시 **`ESC`** 또는 **`F12`** 키를 누릅니다.

## 🛠️ 개발 환경 및 빌드 (개발자용)

### 기술 스택
* **Backend**: Python 3.12+ (Eel, PyAutoGUI, Pyperclip, Keyboard)
* **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
* **Packaging**: PyInstaller

### 로컬 실행 및 EXE 패키징 빌드 방법
1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/semyo0513/Copy_Manager.git
   cd Copy_Manager
   ```
2. 종속 라이브러리를 설치하고 실행 파일을 빌드합니다:
   * 로컬 파이썬 환경의 DLL 모듈 누락 방지 로직이 내장된 `build_exe.py`를 원클릭으로 실행합니다.
   ```bash
   python build_exe.py
   ```
3. 빌드가 성공하면 **`dist/모두의_복붙.exe`** 파일이 단일 패키지로 생성됩니다.
